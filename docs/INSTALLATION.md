# Installation Guide

## Prerequisites

- Linux system (Ubuntu 20.04+ recommended)
- 4GB RAM minimum
- 2GB free disk space

## Quick Setup

### 1. Clone Repository
```bash
git clone https://github.com/YOUR_USERNAME/FreeRTOS-RTOS-Lab-Project.git
cd FreeRTOS-RTOS-Lab-Project
```

### 2. Run Setup Script
```bash
chmod +x scripts/setup.sh
./scripts/setup.sh
```

### 3. Activate Environment
```bash
source env.sh
```
